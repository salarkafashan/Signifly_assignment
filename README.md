## Instalation

This project developed on Docker so for better experience it should be better have Docker on your System.
if you don't have you can download it from [here](https://www.docker.com/get-started).

- Please open your terminal (command prompt on Windows OP).
- type git clone https://github.com/salarkafashan/Signifly_assignment.git
- go to project directory
- type ./vendor/bin/sail up -d (this part take time)
- type ./vendor/bin/sail artisan migrate
- type ./vendor/bin/sail artisan DB:seed

Project successfully instaled.
